Exercício de fixação - HTML

Lista com 10 tags aprendidas em aula:

Tag <p>
A tag p indica que um parágrafo será adicionado.
Tag <h1>
A tag h1 acrescenta um título e dá valor semântico à página.
Tag <title>
Essa tag define o nome que aparecerá na aba da página.
Tag <class>
Atribui uma classe ao elemento, podendo ser utilizada para um ou mais elementos.
Tag <hidden>
Utilizada para ocultar um elemento.
Tag <b>
Transforma o conteúdo em negrito.
Tag <img>
Serve para incluir uma imagem à página.
Tag <ol>
Cria uma lista ordenada.
Tag <ul>
Cria uma lista não ordenada.
Tag <style>
Tag utilizada para incluir código CSS e estilizar nossa página.
